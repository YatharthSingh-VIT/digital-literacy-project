## About This Project

This repository is my submission for the CSE0001 Digital Literacy
project at VIT Bhopal University. The project is built around a
scenario where I am a Student Digital Ambassador helping my
batchmates navigate the digital world — from building professional
profiles to staying safe online.

Each folder in this repo corresponds to one task from the project.
All five tasks are done. This README is a quick overview of
what is inside each one.

---

## Repository Structure

- task-1-presentation — Digital Literacy Infographic
- task-2-portfolio — Screenshots of digital profiles
- task-3-platforms — HackerRank and Google Form screenshots

- task-5-cybercrime — Ransomware case study and prevention checklist

## Task Summaries

### Task 1 — Digital Literacy Infographic
Created a one page infographic using Canva covering five topics —
what digital literacy is, useful digital tools, internet safety,
professional presence and email etiquette. Used a vintage kraft
paper style with a clean alternating layout.

Folder: task-1-presentation/

Set up and updated profiles on three platforms — GitHub, LinkedIn
